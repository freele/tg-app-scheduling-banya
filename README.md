# Баня на даче — Система бронирования

> Для разработчиков: см. [Architecture.md](Architecture.md)

Система бронирования для бани в Португалии. Клиенты бронируют через Telegram, бронирования автоматически синхронизируются с Google Calendar.

## Управление программами

Админ-панель → [События](https://tg-app-scheduling-banya-admin.vercel.app/dashboard/events)

### Редактирование

1. Открыть таблицу программ
2. Найти нужную программу
3. Нажать **Edit**
4. Изменить поля (название, цена, длительность, описание)
5. Нажать **Save**

### Загрузка фотографии

1. Найти программу в таблице
2. Нажать на цветной квадрат с инициалами (или **+**)
3. Выбрать файл (JPEG, PNG)
4. Фото автоматически уменьшится до 800×600

### Включение/отключение

1. Найти программу в таблице
2. Нажать кнопку **Active** / **Off**
3. Отключённые программы не показываются клиентам

## Управление бронированиями

Админ-панель → [Бронирования](https://tg-app-scheduling-banya-admin.vercel.app/dashboard/bookings)

### Просмотр

- Список всех бронирований
- Фильтры по статусу и дате

### Отметка оплаты

1. Найти бронирование в списке
2. Нажать **Mark Paid** или открыть детали
3. В деталях можно указать сумму и способ оплаты

## Calendly

Расписание и доступность настраиваются в [Calendly](https://calendly.com/banya-portugal).

В админ-панели у каждой программы есть ссылка **Manage** — переход в настройки этой программы в Calendly.

### Что настраивается

- Доступные дни и часы
- Буферное время между бронированиями
- Вопросы для клиентов (телефон, количество гостей)
- Синхронизация с Google Calendar

---

## Как это работает

```
Клиент открывает Telegram бота
         ↓
Выбирает программу в Mini App
         ↓
Бронирует через Calendly
         ↓
Бронирование появляется в Google Calendar
         ↓
Бронирование появляется в админ-панели
         ↓
Администратор отмечает оплату
```

---

## Интеграции

- **Calendly** — управление расписанием и доступностью
- **Google Calendar** — автоматическая синхронизация бронирований
- **Supabase** — база данных и хранение фотографий
- **Vercel** — хостинг приложений
